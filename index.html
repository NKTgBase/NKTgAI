NKTg AI is not a summarization tool. It is a specialized Language Decoding System that uses physical algorithms to measure semantic energy and extract the Core Content — the core logical genetic code of a text.

        
1. The Nature of Core Content

        
Core Content is the sentence with the highest density of executable actions and the most concrete outcome in a text. If removed, the text loses key information that cannot be inferred from the rest.

        
It is an original sentence from the source text — not interpreted or altered by subjective reasoning.

        
2. Philosophy

        
Generative AI
Reads and reinterprets a text using the AI's own language.

        
NKTg AI
Measures the energy of each word and sentence to identify core content that already exists. Returns the author's original linguistic genetic code without generating new content.

        
3. Text Structural Genetic Code

        
AMP (Amplifying) + DAMP (Damping) + STABLE = 100%

        
AMP Amplifying > 55%

        
Increasing energy — Actions, Assertions, Execution, Results. Text has a clear focal point; Core Content has high reliability.

        
DAMP Damping > 55%

        
Decreasing energy — Conditions, Context, Risks, Exceptions, Counterarguments. Text tends toward condition analysis or risk assessment.

        
STABLE AMP ≈ DAMP

        
Balanced state — Technical info, Pure data, Data tables, Descriptive content. A prominent Core Content may not exist.

        
4. Output Modes

        
🧠 Left Brain (Extraction) — Distillation

        
Standard
Retains the most important sentences by the Golden Ratio. Best for quick reading.

        
Condensed
Removes repetitive or semantically similar sentences. Best for long texts.

        
Essence
Converges to a single Core Content sentence. Recommended when AMP > 55%.

        
🧠 Right Brain (Addition) — Expansion

        
Refined
Preserves content nucleus with minimum necessary context.

        
Expanded
Core Content combined with surrounding relevant context.

        
Comprehensive (100%)
Full text with DAMPING components marked for easy distinction.

        
5. Value for Experts & Managers

        
    
          
  • Quantifiability: AMP%, DAMP%, Compression%, retained sentences
    • 
          
  • Causality: Core Content often in Action → Result structures
    • 
          
  • Decision-Making: Quickly evaluate whether to read, examine, or act
    • 
          
  • Objectivity: Algorithm-based, not subjective emphasis
    • 
        

        
6. Value for General Users

        
    
          
  • Read Quickly: Identify the highest-action-density sentence instantly
    • 
          
  • Evaluate Before Reading: Observe AMP/DAMP in round 1
    • 
          
  • Objective Comparison: Compare Core Content across multiple texts
    • 
          
  • Identify Key Points: Spot conditions, warnings, or risks
    • 
        

        
7. Limitations

        
If Compression > 90% in Standard or Refined mode, manually copy or upload a file to ensure input integrity.

        
NKTg AI does not perform well on handwritten documents — OCR accuracy drops significantly compared to printed text, which may affect extraction quality.

p = m × v
NKTg₁ = x × p    (Semantic Potential Energy)
NKTg₂ = (dm/dt) × p    (Semantic Kinetic Energy)
NKTg(total) = f(NKTg₁, NKTg₂)

NKTg AI は要約ツールではありません。物理的アルゴリズムを使って意味エネルギーを測定し、コアコンテンツ（Core Content）——テキストの論理的遺伝子コード——を抽出する専門的な言語デコーディングシステムです。

        
1. コアコンテンツの性質

        
コアコンテンツ：実行可能なアクションの密度が最も高い文。削除すると、残りの文から推測できない重要情報が失われます。

        
それは原文からのそのままの一文であり、主観的な解釈や改変は加えられていません。

        
2. 哲学

        
生成 AI
テキストを読み、AI 自身の言語で再解釈します。

        
NKTg AI
各単語とセンテンスのエネルギーを測定し、すでに存在するコアコンテンツを識別。著者の元の言語的遺伝子コードを返します。

        
3. テキスト構造遺伝子コード

        
AMP + DAMP + STABLE = 100%

        
AMP > 55%
エネルギー増加 — アクション、主張、実行、結果。

        
DAMP > 55%
エネルギー減少 — 条件、コンテキスト、リスク、反論。

        
STABLE AMP ≈ DAMP
均衡状態 — 技術情報、純粋なデータ。

        
4. 出力モード

        
🧠 左脳（抽出）— 蒸留

        
Standard
黄金比に従い最重要文を保持。

        
Condensed
繰り返し文を除去。

        
Essence
単一のコアコンテンツに収束。AMP > 55% 推奨。

        
🧠 右脳（追加）— 拡張

        
Refined
コアを保持し最小限のコンテキストを追加。

        
Expanded
コアと関連コンテキストの組み合わせ。

        
Comprehensive
DAMPING コンポーネントをマークした完全テキスト。

        
5. 専門家・マネージャーにとっての価値

        
  • 定量化：AMP%、DAMP%、圧縮率%
  • 因果性：アクション → 結果
  • 意思決定：継続読書・行動の迅速評価
  • 客観性：エネルギー測定アルゴリズム

        
6. 一般ユーザーにとっての価値

        
  • 速読：アクション密度が最も高い文を即座に特定
  • 読む前の評価：最初のラウンドで AMP/DAMP を観察
  • 客観的比較：複数のテキスト間でコアコンテンツを比較
  • 重要ポイントの把握：条件・警告・リスクを発見

        
7. 制限事項

        
Standard または Refined モードで 圧縮率 > 90% の場合、手動でコピーまたはファイルをアップロードしてください。

        
NKTg AI は手書き文書には十分に対応していません —— 印刷文字と比べてOCRの認識精度が大きく低下し、抽出の品質に影響する場合があります。

        
8. NKTg 法則 — 可変慣性アルゴリズム

        
NKTg = f(x, v, m)
p = m × v
NKTg₁ = x × p    (意味的位置エネルギー)
NKTg₂ = (dm/dt) × p    (意味的運動エネルギー)
NKTg(total) = f(NKTg₁, NKTg₂)

      
        

            
すべての処理は WebAssembly によってブラウザ上で直接実行されます。

            
    
                
  • テキストがユーザーのデバイスを離れることはありません
    • 
                
  • サーバーなし、クラウドなし、リモートデータベースなし
    • 
                
  • 履歴は localStorage に保存され、完全にユーザーのデバイス上にあります

NKTg AI — не инструмент для резюмирования. Это специализированная Система декодирования языка, использующая физические алгоритмы для измерения семантической энергии и извлечения основного содержания (Core Content) — ключевого логического генетического кода текста.

        
1. Природа основного содержания

        
Основное содержание: предложение с наибольшей плотностью исполняемых действий. Его удаление лишает текст ключевой информации.

        
Это исходное предложение из исходного текста — без интерпретации или изменения субъективными рассуждениями.

        
2. Философия

        
Генеративный ИИ
Читает и переформулирует текст на собственном языке.

        
NKTg AI
Измеряет энергию каждого слова для идентификации уже существующего основного содержания. Возвращает оригинальный языковой генетический код автора.

        
3. Генетический код структуры текста

        
AMP + DAMP + STABLE = 100%

        
AMP > 55%
Возрастающая энергия — действия, утверждения, результаты.

        
DAMP > 55%
Убывающая энергия — условия, контекст, риски, контраргументы.

        
STABLE AMP ≈ DAMP
Сбалансированное состояние — технические данные, описания.

        
4. Режимы вывода

        
🧠 Левое полушарие (Извлечение)

        
Standard
Сохраняет важнейшие предложения по золотому сечению.

        
Condensed
Удаляет повторяющиеся предложения.

        
Essence
Сводится к одному основному предложению. Рекомендуется при AMP > 55%.

        
🧠 Правое полушарие (Добавление)

        
Refined
Ядро содержания + минимальный контекст.

        
Expanded
Основное содержание + релевантный контекст.

        
Comprehensive
Полный текст с отмеченными компонентами DAMPING.

        
5. Ценность для экспертов и руководителей

        
  • Количественность: AMP%, DAMP%, сжатие%
  • Причинность: Действие → Результат
  • Принятие решений: Быстрая оценка актуальности
  • Объективность: Алгоритм измерения энергии

        
6. Ценность для обычных пользователей

        
  • Быстрое чтение: Мгновенно определить предложение с наибольшей плотностью действий
  • Оценка перед чтением: Наблюдать AMP/DAMP в первом раунде
  • Объективное сравнение: Сравнивать основное содержание разных текстов
  • Выявление ключевых моментов: Замечать условия, предупреждения или риски

        
7. Ограничения

        
Если сжатие > 90% в режиме Standard или Refined: скопируйте вручную или загрузите файл.

        
NKTg AI плохо работает с рукописными документами — точность OCR заметно снижается по сравнению с печатным текстом, что может повлиять на качество извлечения данных.

        
8. Закон NKTg — Алгоритм переменной инерции

        
NKTg = f(x, v, m)
p = m × v
NKTg₁ = x × p    (Семантическая потенциальная энергия)
NKTg₂ = (dm/dt) × p    (Семантическая кинетическая энергия)
NKTg(total) = f(NKTg₁, NKTg₂)
